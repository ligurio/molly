## [0.1.0] - 2022-06-21

Initial version of Jepsen-like framework written in Lua programming language.

### Added

- Add compatibility of Jepsen-compatible format.
- Add support of Tarantool fibers.
- Add support of Lua coroutines.
- Add GH Actions workflows with check, testing, publishing actions.
- Add Luarocks spec.
- Add examples with SQLite tests.
- Add list-append and rw-register operation generators.
